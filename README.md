# sds-drawing-app
A simple web application which can be used to draw simple shapes of your choice of colour and size
- Brush Tool – Freehand drawing with adjustable size and color.
- Shapes – Draw rectangles, circles, and triangles.
- Dark Mode – Toggle between light and dark themes, persists after refresh.
- Color Picker – Choose stroke color dynamically.
- Undo – Step back to previous canvas states using a stack-based history.
- Add Random Image – Insert random images from Picsum.
- Clear Canvas – Reset the canvas completely.
- Persistence – Canvas drawings, dark mode, and brush color are saved in localStorage.
